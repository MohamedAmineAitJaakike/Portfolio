## ðŸ› Troubleshooting

### Build Fails

```bash
# Clear cache and reinstall
rm -rf .next node_modules package-lock.json
npm install
npm run build
```

### Port 3000 in Use (Windows)

```powershell
# Find process using port 3000
netstat -ano | findstr :3000

# Kill the process
taskkill /PID <PID> /F
```

### Image Not Showing

- Verify `public/ma_photo.jpg` exists
- Check file is < 2MB
- Ensure format is JPG or PNG
- Fallback gradient shows if missing

### Slow Performance

- Check Vercel Analytics
- Use Next.js Image component
- Implement code splitting
- Optimize images (< 500KB)
