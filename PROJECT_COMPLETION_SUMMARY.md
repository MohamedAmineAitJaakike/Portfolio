## Design Philosophy

### Color Scheme
```css
Primary Dark:    #0f172a (Slate-950) - Background
Secondary:       #1e293b (Slate-800) - Cards
Accent Blue:     #3b82f6 (Blue-500)  - Primary accent
Accent Cyan:     #06b6d4 (Cyan-500)  - Secondary accent
Text Primary:    #f1f5f9 (Slate-100) - Headings
Text Secondary:  #cbd5e1 (Slate-300) - Body text
```

### Typography
- **Headings**: Bold, 32-56px, high contrast
- **Body**: Regular, 16-18px, excellent readability
- **Accents**: Gradient text for key phrases

### Layout Principles
- Clean whitespace usage
- Clear visual hierarchy
- Responsive breakpoints (mobile-first)
- Maximum 1280px container width
- 16px baseline grid

### Add New Sections

Template for new section:
```typescript
"use client";
import { motion } from "framer-motion";

const NewSection = () => {
  const containerVariants = {
    hidden: { opacity: 0 },
    visible: {
      opacity: 1,
      transition: { staggerChildren: 0.15 }
    }
  };

  const itemVariants = {
    hidden: { opacity: 0, y: 20 },
    visible: { opacity: 1, y: 0, transition: { duration: 0.8 } }
  };

  return (
    <section id="new-section" className="section">
      <motion.div
        className="container mx-auto px-4"
        variants={containerVariants}
        initial="hidden"
        whileInView="visible"
      >
        {/* Your content */}
      </motion.div>
    </section>
  );
};

export default NewSection;
```
